# ODS Spreadsheet Editor для VSCode

[English version](README.md)

Полнофункциональный редактор электронных таблиц OpenDocument (.ods), встроенный прямо в VSCode. Никаких сторонних приложений не требуется.

## Быстрый старт

1. Откройте любой файл `.ods` в VSCode
2. Или щёлкните правой кнопкой на папке в проводнике > **Новая таблица (.ods)** для создания из шаблона

---

## Возможности

### Редактирование ячеек
- **Прямой ввод** — нажмите на ячейку и начинайте вводить, или нажмите `F2` / `Enter` для перехода в режим редактирования
- **Строка формул** — редактируйте значение или формулу в верхней панели
- **Поддержка формул** — `=СУММ(A1:A10)`, `=ЕСЛИ(A1>0;"да";"нет")`, `=ВПР(...)` и более 80 функций
- **Автодополнение формул** — введите `=` и начало имени функции; появится выпадающий список совпадений; навигация `↑`/`↓`, принятие `Tab` или `Enter`
- **Ссылки на другие листы** — `=Лист2.A1` или `=Лист2.A1:B10` (нотация ODS с точкой)
- **Автозаполнение** — тяните маркер заполнения (маленький квадрат в нижнем правом углу выделения) для продолжения рядов: числа, буквы, IP-адреса, повторяющиеся шаблоны
- **Перетаскивание ячеек** — удерживайте `Alt` и перетащите выделение, чтобы переместить содержимое в новое место
- **Буфер обмена** — `Ctrl+C`, `Ctrl+V`, `Ctrl+X` в формате TSV (совместим с вставкой из Excel / Google Sheets)

### Выделение и навигация
- **Клик** по ячейке — выбрать её
- **Shift+клик** или **Shift+стрелка** — расширить выделение
- **Клик на заголовок столбца** (A, B, C...) — выбрать весь столбец
- **Клик на заголовок строки** (1, 2, 3...) — выбрать всю строку
- **Ctrl+A** — выбрать все ячейки
- **Tab** / **Shift+Tab** — вправо / влево
- **Enter** — подтвердить ввод и перейти вниз
- **Объединённые ячейки** — клик на любую часть объединённой ячейки выделяет всю область

### Панель форматирования
| Кнопка | Действие |
|--------|----------|
| **Ж** / **К** / **Ч** | Жирный, Курсив, Подчёркнутый |
| **З̶** (Зачёркнутый) | Зачёркнутый текст |
| Выбор цвета | Цвет текста / Цвет фона |
| Выравнивание | По левому краю / По центру / По правому краю |
| Раскрывающийся список границ | Все границы, Внешние, Убрать, Верх/Низ/Лево/Право |
| **123** (Формат числа) | Общий, Число (1 234,56), Валюта ($1 234,56), Процент (12,34%), Дата, Целое |
| Значок переноса текста | Переключить перенос текста в ячейках |
| Значок закрепления | Закрепить / открепить строки и столбцы |
| **А↓** / **Я↓** | Сортировка по возрастанию / убыванию |
| Значок фильтра | Открыть выпадающий список фильтра для выбранного столбца |
| Значок условного форматирования | Открыть диалог условного форматирования |

### Границы
Нажмите кнопку границ на панели инструментов, чтобы открыть раскрывающийся список:
- **Все границы** — применить границы ко всем сторонам ячеек в выделении
- **Внешние границы** — только граница вокруг периметра выделения
- **Убрать границы** — удалить все границы из выделения
- **Одна сторона** — только Верх, Низ, Лево или Право

### Форматы чисел
Нажмите **123** на панели инструментов:
- **Общий** — без форматирования (исходное значение)
- **Число** — `1 234,56` (разделитель тысяч, 2 знака после запятой)
- **Валюта** — `$1 234,56`
- **Процент** — `12,34%` (значение умножается на 100)
- **Дата** — формат `2024-01-15`
- **Целое** — `1 235` (разделитель тысяч, без дробной части)

### Перенос текста
Включите перенос текста, чтобы содержимое ячейки разбивалось на несколько строк. В сочетании с вертикальным выравниванием (верх/центр/низ) позволяет управлять отображением длинного текста.

### Закрепление областей
1. Выберите ячейку на пересечении — всё **выше** и **левее** неё будет закреплено
2. Нажмите кнопку закрепления (значок с пунктирным крестом)
3. Закреплённые строки/столбцы остаются на месте при прокрутке
4. Нажмите ещё раз для открепления

### Изменение размеров столбцов и строк
- **Перетащите** разделитель заголовка для изменения размера столбцов или строк
- **Двойной клик** на разделитель заголовка столбца для **автоподбора** ширины (сканирует все ячейки и подстраивается под самое длинное содержимое)

### Найти и заменить
- **Ctrl+F** — открыть строку поиска
- **Ctrl+H** — открыть строку поиска (с полем замены)
- Вводите текст для поиска по всем ячейкам; совпадения подсвечиваются
- **Enter** / стрелки для перехода между совпадениями
- **Заменить** / **Заменить всё** для подстановки значений
- **Esc** для закрытия

### Фильтр
1. Выберите ячейку в столбце, который нужно отфильтровать
2. Нажмите кнопку фильтра (значок воронки) на панели инструментов
3. Появится раскрывающийся список с **флажками для каждого уникального значения** в столбце
4. Снимите флажки со значений, которые нужно скрыть
5. **Применить** для фильтрации, **Сбросить** для удаления фильтра, **Отмена** для закрытия
6. Отфильтрованные (скрытые) строки пропускаются при отрисовке

### Условное форматирование
1. Выберите диапазон ячеек для применения правила
2. Нажмите кнопку условного форматирования (значок с цветными квадратами)
3. Выберите тип условия:

   **Правила на основе значений:**
   - **Больше** / **Меньше** — сравнение значения ячейки с числом
   - **Равно** / **Не равно** — точное совпадение
   - **Содержит** / **Не содержит** — поиск подстроки
   - **Между** — значение находится в диапазоне (два значения)
   - **Пустая** / **Не пустая** — определение пустых ячеек
   - Задайте стиль форматирования: цвет фона, цвет текста, жирный, курсив

   **Визуальные шкалы:**
   - **Цветовая шкала** — ячейки закрашиваются градиентом от минимального до максимального цвета (опционально с промежуточным); цвет пропорционален значению в диапазоне
   - **Полоса данных** — цветная полоса заполняет ячейку пропорционально её значению; длина полосы отражает относительную величину

4. Нажмите **Применить** — правило сохраняется, ячейки обновляются немедленно
5. Существующие правила перечислены в нижней части диалога с кнопкой удаления (×)
6. Несколько правил могут накладываться; применяется первое совпадающее правило

### Комментарии к ячейкам
- **Добавить/изменить комментарий**: щёлкните правой кнопкой на ячейке > **Редактировать комментарий**, введите текст в диалоге, нажмите **Сохранить**
- **Удалить комментарий**: откройте диалог комментария > **Удалить**
- **Просмотр комментария**: наведите курсор на ячейку с красным треугольником (верхний правый угол) — появится всплывающая подсказка с текстом
- Комментарии сохраняются в файле `.ods` и совместимы с LibreOffice/Google Sheets

### Сортировка
1. Выберите диапазон ячеек
2. Нажмите **А↓** для сортировки по возрастанию или **Я↓** по убыванию
3. Сортирует выбранный диапазон по активному столбцу

### Объединение ячеек
Щёлкните правой кнопкой на выделении > **Объединить ячейки** для объединения нескольких ячеек в одну. **Разъединить** для разделения обратно.

### Вставка и удаление
Щёлкните правой кнопкой мыши для доступа к:
- **Вставить строку выше / ниже**
- **Вставить столбец слева / справа**
- **Удалить строку / столбец**

### Управление листами
- **Вкладки листов** внизу отображают все листы
- Нажмите на вкладку для переключения
- **Двойной клик** на вкладке для переименования
- **Правый клик** на вкладке для удаления
- Нажмите **+** для добавления нового листа

### Строка состояния
При выборе нескольких ячеек в правом нижнем углу отображается:
- **Сумма** — сумма всех числовых значений
- **Среднее** — среднее значение
- **Кол-во** — количество непустых ячеек

## Импорт / Экспорт CSV

### Экспорт в CSV
Откройте палитру команд (`Ctrl+Shift+P`) > **ODS: Export Active Sheet to CSV**

Сохраняет активный лист как файл `.csv` (стандарт RFC-4180). Поля, содержащие запятые, кавычки или переносы строк, автоматически заключаются в кавычки.

### Импорт из CSV
- Щёлкните правой кнопкой на любом файле `.csv` в проводнике > **Import CSV as ODS**
- Или откройте `Ctrl+Shift+P` > **ODS: Import CSV as ODS**

Создаёт новый файл `.ods` в той же папке и открывает его в редакторе. Числа определяются автоматически и сохраняются как числовые значения.

## Шаблоны

### Встроенные шаблоны
При создании новой таблицы (правый клик на папке > **Новая таблица**) выберите из:

| Шаблон | Описание |
|--------|----------|
| **Пустой** | Пустая таблица |
| **Бюджет** | Ежемесячный бюджет с категориями доходов/расходов, 12 месяцев, формат валюты |
| **Счёт-фактура** | Макет счёта с реквизитами, позициями, количеством, суммами, итогом |
| **Табель** | Еженедельный табель с днями, временем начала/конца, перерывом, часами |
| **Контакты** | Адресная книга с именем, email, телефоном, компанией, адресом |
| **Трекер задач** | Список задач с №, задачей, исполнителем, приоритетом, статусом, сроком |

### Пользовательские шаблоны
Сохраните любую таблицу как многоразовый шаблон:

1. Откройте файл `.ods`, который хотите использовать как шаблон
2. Откройте палитру команд (`Ctrl+Shift+P`) > **ODS: Save Current as Template**
3. Введите имя шаблона и необязательное описание
4. Шаблон появится в списке выбора **Новой таблицы** в разделе «Пользовательские шаблоны»

Пользовательские шаблоны хранят полный файл `.ods` (данные, стили, формулы, форматирование) и доступны во всех рабочих пространствах.

## Формулы

### Автодополнение формул
При вводе формулы, начинающейся с `=`, по мере набора имени функции появляется выпадающий список совпадений. Используйте `↑`/`↓` для навигации, `Tab` или `Enter` для вставки функции (курсор размещается внутри скобок), `Esc` для закрытия.

### Поддерживаемые функции

**Математические и статистические:** `SUM`, `SUMIF`, `SUMIFS`, `SUMPRODUCT`, `AVERAGE`, `AVERAGEIF`, `AVERAGEIFS`, `MIN`, `MAX`, `COUNT`, `COUNTA`, `COUNTBLANK`, `COUNTIF`, `COUNTIFS`, `ABS`, `ROUND`, `ROUNDUP`, `ROUNDDOWN`, `CEILING`, `FLOOR`, `MOD`, `POWER`, `SQRT`, `PI`, `RAND`, `RANDBETWEEN`, `INT`, `SIGN`, `LOG`, `LOG10`, `LN`, `EXP`

**Финансовые:** `PMT`, `PV`, `FV`, `NPV`, `IRR`, `RATE`, `NPER`, `SLN`

| Функция | Описание |
|---------|----------|
| `PMT(ставка; кпер; пс; [бс]; [тип])` | Периодический платёж по кредиту |
| `PV(ставка; кпер; плт; [бс]; [тип])` | Текущая стоимость инвестиции |
| `FV(ставка; кпер; плт; [пс]; [тип])` | Будущая стоимость инвестиции |
| `NPV(ставка; значение1; ...)` | Чистая приведённая стоимость денежных потоков |
| `IRR(значения; [предположение])` | Внутренняя норма доходности |
| `RATE(кпер; плт; пс; [бс]; [тип]; [предположение])` | Процентная ставка за период |
| `NPER(ставка; плт; пс; [бс]; [тип])` | Количество периодов выплат |
| `SLN(стоимость; ликвидация; период)` | Линейная амортизация |

**Текстовые:** `LEN`, `LEFT`, `RIGHT`, `MID`, `UPPER`, `LOWER`, `TRIM`, `CONCATENATE`, `SUBSTITUTE`, `FIND`, `SEARCH`, `REPLACE`, `REPT`, `TEXT`, `VALUE`, `EXACT`, `T`

**Логические:** `IF`, `AND`, `OR`, `NOT`, `IFERROR`, `ISBLANK`, `ISNUMBER`, `ISTEXT`, `TRUE`, `FALSE`

**Ссылки:** `VLOOKUP`, `HLOOKUP`, `INDEX`, `MATCH`, `CHOOSE`, `ROW`, `COLUMN`, `ROWS`, `COLUMNS`

**Дата и время:** `TODAY`, `NOW`, `DATE`, `YEAR`, `MONTH`, `DAY`

### Синтаксис ссылок
- Ячейка: `A1`, `$A$1` (абсолютная)
- Диапазон: `A1:B10`
- Другой лист: `Лист2.A1`, `Лист2.A1:B10`
- ODS-скобки: `[.A1]`, `[.Лист2.A1]`

## Клавиатурные сочетания

| Сочетание | Действие |
|-----------|----------|
| `F2` или `Enter` | Начать редактирование ячейки |
| `Esc` | Отменить редактирование |
| `Tab` / `Shift+Tab` | Вправо / влево |
| `Enter` / `Shift+Enter` | Вниз / вверх |
| Клавиши со стрелками | Навигация по ячейкам |
| `Shift+стрелка` | Расширить выделение |
| `Ctrl+A` | Выбрать все |
| `Ctrl+C` / `Ctrl+V` / `Ctrl+X` | Копировать / Вставить / Вырезать |
| `Ctrl+B` | Жирный |
| `Ctrl+I` | Курсив |
| `Ctrl+U` | Подчёркнутый |
| `Ctrl+F` / `Ctrl+H` | Найти / Найти и заменить |
| `Delete` / `Backspace` | Очистить содержимое ячейки |
| `Home` | Перейти к столбцу A в текущей строке |
| `Ctrl+Home` | Перейти к ячейке A1 |
| `End` | Перейти к последней использованной ячейке в строке |
| `Ctrl+End` | Перейти к последней использованной ячейке на листе |
| `Page Up` / `Page Down` | Прокрутить на страницу вверх / вниз |
| `Alt`+перетаскивание | Переместить выделение в новое место |
| Адресная строка + `Enter` | Перейти по введённому адресу (например, `B5`, `A1:C10`) |

## Формат файла

Расширение работает с файлами **OpenDocument Spreadsheet (.ods)** — открытый стандарт, используемый LibreOffice Calc, Google Sheets (экспорт) и другими приложениями. Файлы можно свободно обмениваться между этим редактором и любым ODS-совместимым приложением.
